MNE-Features
============

This is a library proposes Python code for feature extraction with M/EEG data.

Installation
------------

Via ``pip``, the Python package manager::

	$ pip install mne-features

or via ``conda``::

	$ conda install --channel=conda-forge mne-features


Bug reports
-----------

Use the `github issue tracker <https://github.com/mne-tools/mne-features/issues>`_ to report bugs.
